https://college-match-maker.herokuapp.com/ [GET]

Will return the home page of the site.

https://college-match-maker.herokuapp.com/college [POST]

Add a college to the database.

Example of json being sent:

{{l}}
"name": "umass",
"state": "MA",
"students": 5000,
"tuition": 5000,
"rank": 1
{{r}}

https://college-match-maker.herokuapp.com/colleges [GET]

Will return the page that displays all the colleges in the database.

https://college-match-maker.herokuapp.com/getColleges [GET]

Will return json data of all the college in the database.

https://college-match-maker.herokuapp.com/college/{{l}}theCollege{{r}} [GET]

Will return the json data for one college.

The theCollege is a placeholder for the actual college

https://college-match-maker.herokuapp.com/college/{{l}}theCollege{{r}} [DELETE]

Will delete a certain college in the database.

The theCollege is a placeholder for the actual college

https://college-match-maker.herokuapp.com/addPost [POST]

Will add a post to a certain colleges form

Example of json being sent:

{{l}}
"schoolName": "umass",
"author": "Brett",
"Topic": "GPA",
"message": "What is the average GPA to go to this school?"
{{r}}

https://college-match-maker.herokuapp.com/sendCollegeInfo [POST]

Will return a json array of suggested colleges

Example of json being sent:

{{l}}
"size": 5000,
"state": "Arizona",
"tuition": 5000,
{{r}}
